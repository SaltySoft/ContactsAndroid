ContactsAndroid
===============

Application Android "Gestionnaire de Contact" SaltyContacts.
Application réalisée par Vincent Lefebvre, Antoine Jackson 
et Fabien Bousquet. 

Fonctionnalités obligatoires :
- Afficher la liste des contacts 
- Ajouter un contact 
- Modifier un contact 
- Visualiser un contact 
- Supprimer un contact 
- Création de TAG personnalisé.
- Un clic sur le numéro de téléphone d’un 
  contact redirige vers l’application 
  permettant de composer un numéro de 
  téléphone avec le numéro du contact pré rempli 
- Un clic sur une adresse mail, ouvre une 
  application Email avec le destinataire pré 
  rempli avec l’adresse cliquée 
- Un clic sur l’adresse postale ouvre 
  l’application Google Maps avec position 
  sur l’adresse en question. 
- Un clic sur l’emplacement de la photo, 
  permet de prendre une nouvelle photo 
  ou de sélectionner une photo depuis la 
  galerie afin de l’assigner au contact 

Fonctionnalités bonus :
- Base de donnée persistente.
- Connexion avec la base de donnée de contact d'Android. 
  Il est possible d'importer les contacts du téléphone 
  dans la base de donnée de l'application SaltyContacts.
  Dans la vue d'affichage de la liste des contacts,
  en allant dans settings > Import Contacts.
